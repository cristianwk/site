Configurar entrega de e-mails

Alteração dos registros de troca de e-mail (MX)

Os registros de troca de e-mails (MX) controlam como os e-mails de entrada são roteados para o seu domínio. Para que o Google hospede seu e-mail, você precisa alterar esses registros MX para direcioná-los a nossos servidores.

Seus registros MX atuais contêm valor(es) incorreto(s):

Endereço do servidor MX Prioridade
apmx.l.google.com. 0
Para corrigir esse problema, digite os registros MX corretos, como mostrado nas instruções completas abaixo.

Se o seu domínio já possuir endereços de e-mail, tenha cuidado ao alterar registros MX. Para evitar interrupção do serviço de e-mail, certifique-se de criar o mesmo conjunto de contas de usuário no painel de controle antes de alterar os registros MX. Saiba mais

Se você ainda não estiver pronto para alterar seus registros MX, use o endereço de e-mail temporário (<nomedousuário>@awktec.com.test-google-a.com) atribuído às suas contas de usuário quando as criou. O endereço de e-mail temporário será desativado quando você alterar os registros MX. Não deixe de confirmar a propriedade de domínio antes de usar quaisquer aplicativos do Google Apps.

Exibição de instruções para

  1. Acesse o site da empresa de hospedagem de domínio utilizando o nome de usuário e a senha associados ao seu domínio.

  2. Navegue para a página de manutenção de um registro MX. Os registros MX são registros DNS (Domain Name Service) especiais, e freqüentemente estão localizados em seções intituladas "DNS Management", "Mail Server Configuration" ou "Name Server Management". Talvez seja necessário ativar as configurações avançadas para permitir a edição desses registros MX.

  3. Exclua quaisquer registros MX existentes antes de inserir novos registros MX.

  4. Para cada registro MX, digite as informações de acordo com as entradas na tabela a seguir.

    Talvez não seja possível informar os valores prioritários exatamente como aparecem na tabela a seguir; nesse caso, simplesmente certifique-se de que os endereços do servidor tenham prioridade na mesma ordem que aparecem na tabela (ou seja, a classificação de prioridade [1, 3, 3, 5, 5, 5, 5] deverá funcionar da mesma forma que [1, 5, 5, 10, 10, 10, 10] contanto que mantenha os endereços na ordem correta).

    Se for solicitado que você especifique o tipo de cada registro que está adicionando, digite "MX".

    Os registros MX com freqüência exigem o formato específico de registros DNS, incluindo um ponto final (".") ao fim de quaisquer nomes de domínio totalmente qualificados (por exemplo, "server.example.com").

    Defina os valores para o valor máximo permitido.
